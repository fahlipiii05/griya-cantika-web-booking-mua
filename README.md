# Sistem Booking Make Up Artist
Sistem booking Make Up Artist berbasis web yang dikembangkan sebagai proyek tugas akhir perkuliahan.

## Fitur
- Manajemen layanan (CRUD)
- Sistem booking online
- Manajemen jadwal
- Dashboard admin

## Teknologi yang Digunakan
- PHP
- CodeIgniter 4
- MySQL
- Bootstrap

## Jenis Proyek
Proyek Akademik / Tugas Akhir

## Catatan
Proyek ini dikembangkan untuk keperluan pembelajaran dan akademik.
